# Wandao v1.3.6

本版本修复知识星球 Group 精华帖入口与断点续传，并调整平台页结果展示，避免悬浮区域遮挡表单。

## 主要变化

- 正确识别 `/digests/<group_id>`，按精华帖 API 导出，而非误作普通链接页。
- API 明确关联完整文章的帖子会以同一 `topicId` 原位升级预览 Markdown；普通正文链接仍独立导出，不会误覆盖。
- 知识星球 Group 导出支持在高级选项中设置长休眠起始篇数、间隔、最短与最长时长。
- 恢复任务优先处理已有断点，避免先检查新帖触发不必要的风控等待。
- 平台页的操作区、进度和任务结果不再悬浮遮挡表单。
