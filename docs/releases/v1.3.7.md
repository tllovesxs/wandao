# Wandao v1.3.7

这是针对 v1.3.6 桌面安装包无法启动的紧急热修版本。

## 修复

- 修复 Windows 与 macOS 安装包遗漏 `guide_assets.js`，导致主进程启动时报 `Cannot find module './guide_assets'` 的问题。
- 增强 Windows 解包安装包冒烟检查：检查 `app.asar` 中是否包含 Electron 主进程及其静态本地 CommonJS 依赖，避免源码存在但未被白名单打包的情况再次进入发行版。

## 升级说明

- 如果已下载 v1.3.6，请改为下载并覆盖安装 v1.3.7；无需清除应用数据或重新登录。
