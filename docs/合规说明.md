# 合规说明

万能导用于把用户已经有权限访问的知识库内容导出到本地，减少手动复制粘贴的重复劳动。

## 工具边界

万能导：

- 不破解登录。
- 不绕过权限控制。
- 不提供未授权内容访问能力。
- 不移除平台水印或版权声明。
- 不建议高频、并发或批量滥用。

工具通过浏览器自动化模拟用户正常打开页面、读取目录、复制正文和保存 Markdown 的流程。默认加入约 1 秒的请求间隔，并提供随机浮动和停止按钮，用于降低高频请求风险。

## 使用者责任

使用者需要自行确认：

- 自己有权限访问目标内容。
- 导出、保存、分享、再发布行为符合平台服务条款。
- 导出内容的使用符合版权、团队制度和所在地法律法规。
- 不将导出内容用于未授权传播、售卖、公开发布或训练数据收集。

## 推荐使用方式

- 仅导出自己创建、购买、加入团队后被授权访问的内容。
- 保留合理请求延迟，不把延迟调成 0。
- 遇到平台频控、验证码或访问异常时停止任务。
- 不公开提交 Cookie、浏览器配置目录、导出文件和私有知识库内容。

如果你不确定某项使用是否被允许，请先咨询内容权利方或平台规则。
